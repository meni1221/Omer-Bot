# 🕯️ Omer-Bot: Smart AI-Driven WhatsApp Automation
**An automated daily reminder system for "Sefirat HaOmer" featuring dynamic "Zmanim" scheduling and custom AI art.**

## 🚀 Overview
Omer-Bot brings a high-end cinematic experience to the daily Omer count. Every evening, **exactly at Tzeit HaKochavim (dusk)**, the bot delivers:
- **Handcrafted AI Art:** 49 unique 8K cinematic renders, manually curated for each day of the count.
- **Real-time Zmanim:** Dynamic scheduling based on daily astronomical calculations (Bnei Brak location).
- **Interactive Content:** Hebrew counting text and community engagement links.

## ✨ Key Features
- **Dynamic Scheduling:** Uses the Hebcal API to fetch precise daily "Zmanim" and trigger the message at the right halachic time.
- **Manual Art Direction:** Each of the 49 images was manually crafted using advanced AI workflows to ensure a consistent cinematic aesthetic.
- **Headless Cloud Support:** Fully optimized for Render.com and other cloud providers.
