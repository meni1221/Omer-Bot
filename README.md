# 🕯️ Omer-Bot: Smart AI-Driven WhatsApp Automation
**An automated daily reminder system for "Sefirat HaOmer" featuring dynamic "Zmanim" scheduling and custom AI art.**

## 🚀 Overview
Omer-Bot brings a high-end cinematic experience to the daily Omer count. The system fetches daily astronomical data to ensure the message is delivered exactly when needed.

## ✨ Key Features
- **Dynamic Scheduling:** Uses the Hebcal API to trigger messages exactly at **Tzeit HaKochavim** (dusk) each day.
- **Friday/Shabbat Logic:** Smart built-in logic that detects Fridays and sends the reminder **early (before Shabbat starts)** to ensure users receive the update on time.
- **Handcrafted AI Art:** 49 unique 8K cinematic renders, manually curated for each day of the count to provide a premium user experience.
- **Headless Cloud Support:** Fully optimized for cloud providers like Render.com.
